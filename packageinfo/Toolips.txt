Toolips is a *extensible*, **parametric** networking suite for 
the Julia programming language primarily centered around web-development. 
This project was created to tackle **any** 
web-development task in Julia. The framework supports full-stack web-development,
API creation, and even UDP and other HTTP servers like proxy servers.
